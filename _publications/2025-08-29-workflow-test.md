這個檔案用於測試 GitHub Actions workflow 是否能正常觸發和執行。

## 測試目標

1. 驗證 workflow 能被觸發
2. 驗證自動時間戳功能
3. 驗證網站能正常建置和部署

如果您看到這篇文章顯示在網站上，表示 workflow 運作正常！
